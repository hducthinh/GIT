# Lệnh git clone

Lệnh `git clone` được sử dụng để sao chép (clone) một repository có sẵn từ remote (ví dụ: GitHub, GitLab) về máy tính cá nhân (local).

## Cú pháp cơ bản
```bash
git clone <url-của-repository>
```

## Cách sử dụng
- Khi chạy lệnh này, Git sẽ tự động tạo ra một thư mục mới có tên giống với tên repository, tải toàn bộ mã nguồn, các nhánh và lịch sử commit về máy của bạn.
- Lệnh này thường chỉ sử dụng một lần khi bạn bắt đầu tham gia vào một dự án đã có sẵn.

## Ví dụ
```bash
git clone https://github.com/user/project.git
```

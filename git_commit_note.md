# Lệnh git commit

Lệnh `git commit` được sử dụng để ghi lại (lưu lại) những thay đổi hiện tại trong staging area vào lịch sử của kho chứa (repository). Mỗi commit đóng vai trò như một điểm lưu (snapshot) giúp bạn có thể xem lại hoặc quay về sau này.

## Tác dụng chính
- Lưu lại một phiên bản của dự án tại một thời điểm nhất định.
- Đi kèm với một thông điệp (message) để mô tả những thay đổi bạn vừa thực hiện.
- Giúp theo dõi quá trình phát triển và cộng tác hiệu quả hơn.

## Cú pháp
```bash
git commit -m "Thông điệp mô tả thay đổi của bạn"
```

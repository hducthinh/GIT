# Lệnh git log

Lệnh `git log` cho phép bạn xem lại lịch sử các lần commit đã thực hiện trong kho chứa (repository) của bạn. Nó như một cuốn nhật ký ghi lại toàn bộ quá trình thay đổi mã nguồn.

## Tác dụng chính
- Xem danh sách các commit với thông tin chi tiết: mã hash, tác giả, thời gian, và thông điệp commit.
- Theo dõi quá trình phát triển dự án hoặc tìm kiếm thời điểm xảy ra một lỗi cụ thể.
- Kiểm tra các commit gần nhất trên nhánh hiện tại.

## Cú pháp
Xem lịch sử chi tiết:
```bash
git log
```
Xem lịch sử dạng tóm tắt trên một dòng:
```bash
git log --oneline
```

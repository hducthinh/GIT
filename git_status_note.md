# Lệnh git status

Lệnh `git status` giúp bạn kiểm tra trạng thái hiện tại của working directory và staging area trong dự án. Đây là lệnh thường xuyên được sử dụng nhất để biết mình đang ở đâu và đã làm gì.

## Tác dụng chính
- Hiển thị nhánh hiện tại bạn đang làm việc.
- Hiển thị những file nào đã bị thay đổi (modified) nhưng chưa được thêm vào staging.
- Hiển thị những file nào đã được đưa vào staging area (staged) để chuẩn bị cho lần commit tiếp theo.
- Hiển thị những file mới hoàn toàn chưa được Git theo dõi (untracked files).

## Cú pháp
```bash
git status
```
